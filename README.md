🎣 **1. 自首：標題黨與販賣焦慮**
開頭寫得聳動純粹為了對抗演算法、求擴散。如果讓您感到不適，先說聲抱歉，請關注內容本身。

🛠️ **2. 共建：歡迎 PR 校準**
若發現嚴重錯誤或認為不合適之處，與其單純批評，更歡迎透過 PR (Pull Request) 直接協助校準。我會定期維護，並由衷感謝每一位願意實際動手優化內容的貢獻者。

🧘 **3. 心態：別對商業邏輯應激**
獨立遊戲與商業遊戲雖脈絡不同，但「商業邏輯」客觀存在且具參考價值。建議放下成見，理性看待其中的設計思維。

💌 **4. 初衷：寫給入行前的自己**
這是一份我希望當初能看到的「新人避坑指南」。目的很單純：幫你快速搞懂基礎概論，少走冤枉路。

🏭 **5. 視角：大廠的工業化路數**
內容多源於大廠的工業化流程。這只是遊戲開發的「其中一種解法」（偏商業邏輯），非標準答案，請取其精華。


### 🕳️ 兔子洞裡有什麼？

本書不談情懷，只談**控制**。
它揭露了那些讓 30 億人像小白鼠一樣點擊屏幕的**殘酷真相**：

* 為什麼你的遊戲好玩卻沒人付費？因為你沒學會**販賣焦慮**。
* 為什麼垃圾遊戲能月入過億？因為他們掌握了**合法成癮**的化學方程式。
* 為什麼獨立遊戲必死？因為你試圖用**靈魂**去對抗別人的**工業化收割機**。

這裡沒有雞湯。只有關於**掠奪注意力**、**操縱決策**與**構建數位監獄**的冷血邏輯。

### ⚡ 做出選擇

你有兩個選擇：

1.  關掉這個頁面。繼續相信遊戲是關於快樂與夢想的，做一個幸福的創作者（然後默默消亡）。
2.  **下載這份文件。** 承認遊戲是關於人性弱點的精確計算，拿起這把手術刀，成為遊戲世界的**造物主**。

**但請記住：一旦你打開它，就再也回不去了。**
