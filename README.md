# 遊戲研發與發行全書 (The Game Dev's Black Book)

![License](https://img.shields.io/badge/license-CC%20BY--SA%204.0-green)
![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen)

**別用愛發電！這是將遊戲設計從藝術升維為科學的公式化實戰手冊。**

本書揭示了行業巨頭用來鎖死玩家的底層邏輯，核心是 **「用戶導向逆向推導」**（User-Centric Reverse Deduction）。我們不談空泛的創意，只談可被驗證的 **心理學模型**、**數值結構** 與 **變現邏輯**。
